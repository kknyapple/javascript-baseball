## **📄** 기능 목록

- 컴퓨터 수 생성
  - 숫자 3개 (예외 처리)
  - 1-9까지 수 (예외 처리)
  - 서로 다른 수 (예외 처리)
- 숫자 입력 받기
  - 숫자 3개 (예외 처리)
  - 1-9까지 수 (예외 처리)
  - 서로 다른 수 (예외 처리)
- 게임 하기
  - 같은 수가 같은 자리에 있으면 스트라이크
  - 같은 수가 다른 자리에 있으면 볼
  - 같은 수가 없으면 낫싱
  - 3 스트라이크면 정답
- 게임 종료
  - 정답이면 재시작 선택
  - 1 입력 받으면 새로 시작, 2 입력 받으면 종료
- 게임 새로 시작
  - 새로운 컴퓨터 수 만들기
  - 숫자 입력 받기
